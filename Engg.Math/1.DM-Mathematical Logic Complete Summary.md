## MODULE 1: PROPOSITIONAL LOGIC OPERATORS

### Key Concepts
* **CONJUNCTION (AND / $\wedge$):** True only if both statements are True.
* **DISJUNCTION (OR / $\vee$):** True if at least one statement is True.
* **BICONDITIONAL (IF AND ONLY IF / $\leftrightarrow$):** True only if both statements have the exact same truth value ($T \leftrightarrow T$ or $F \leftrightarrow F$).
* **IMPLICATION (IF-THEN / $\rightarrow$):** This evaluates to False in **ONLY ONE** scenario: when a True premise leads to a False conclusion ($T \rightarrow F = F$). In all other cases, it is mathematically True.
  * *Equivalence Rule:* $P \rightarrow Q$ is logically equivalent to $\neg P \vee Q$.

### The Implication Trap ($P \rightarrow Q$)
Think of $P \rightarrow Q$ as a strict database business rule: *"If a user is an Admin ($P$), then they must have Write Access ($Q$)."*
* **Admin WITH Write Access ($T \rightarrow T$):** Valid state. (True)
* **Admin WITHOUT Write Access ($T \rightarrow F$):** **SYSTEM ERROR.** The condition is met, but the requirement is missing. This is the ONLY time the rule evaluates to False.
* **Standard User WITH Write Access ($F \rightarrow T$):** Valid state. The rule only explicitly restricts Admins; it doesn't forbid standard users from having write access. (True)
* **Standard User WITHOUT Write Access ($F \rightarrow F$):** Valid state. (True)

## MODULE 2: LOGICAL EQUIVALENCES & TRUTH TABLES

### Key Concepts
* **DE MORGAN'S LAWS:** Distributing a negation ($\neg$) flips the internal operators.
  * $\neg(P \vee Q) \equiv \neg P \wedge \neg Q$
  * $\neg(P \wedge Q) \equiv \neg P \vee \neg Q$
* **DISTRIBUTIVE LAWS:** Similar to algebraic multiplication.
  * $P \wedge (Q \vee R) \equiv (P \wedge Q) \vee (P \wedge R)$
  * $P \vee (Q \wedge R) \equiv (P \vee Q) \wedge (P \vee R)$

## MODULE 3: TAUTOLOGY, CONTRADICTION, AND CONTINGENCY

### Key Concepts
* **TAUTOLOGY:** A compound proposition that is always True, regardless of the individual truth values of its variables. (e.g., $P \vee \neg P$).
* **CONTRADICTION:** A compound proposition that is always False. (e.g., $P \wedge \neg P$).
* **CONTINGENCY:** A proposition that is neither a tautology nor a contradiction; its truth value depends on the inputs.

## MODULE 4: FIRST-ORDER LOGIC (QUANTIFIERS)

### Key Concepts
* **UNIVERSAL QUANTIFIER ($\forall x$):** "For all $x$". The rule applies to every single element in the domain.
* **EXISTENTIAL QUANTIFIER ($\exists x$):** "There exists an $x$". The rule applies to at least one element in the domain.
* **NEGATING QUANTIFIERS:** * Negating a "For All" creates an "Exists": $\neg(\forall x, P(x)) \equiv \exists x, \neg P(x)$
  * Negating an "Exists" creates a "For All": $\neg(\exists x, P(x)) \equiv \forall x, \neg P(x)$
* **THE IMPLICATION NEGATION TRAP:** * How to negate $\forall x (P(x) \rightarrow Q(x))$
  * Result: **$\exists x (P(x) \wedge \neg Q(x))$**
  * *Logic:* To disprove "All P's are Q's", you do not say "No P's are Q's". You just need to find "At least one P that is NOT Q".

## DAILY VERIFICATION Q&A (COMBINED QUESTION BANK)

**Q1. Which of the following is the truth value of $(P \rightarrow Q)$ when $P$ is False and $Q$ is True?**
* **Answer:** True 
* **Logic:** An implication is only false if True leads to False. False leading to True does not violate the implication rule.

**Q2. According to De Morgan's Law, the expression $\neg(P \vee Q)$ is logically equivalent to:**
* **Answer:** $\neg P \wedge \neg Q$ 
* **Logic:** The negation distributes to both variables, and the OR ($\vee$) flips to an AND ($\wedge$).

**Q3. The compound proposition $P \vee \neg(P \wedge Q)$ is a:**
* **Answer:** Tautology 
* **Logic:** By De Morgan's, this becomes $P \vee (\neg P \vee \neg Q)$. Grouping the P's gives $(P \vee \neg P) \vee \neg Q$. Since $(P \vee \neg P)$ is always True, the entire statement evaluates to True regardless of Q.

**Q4. What is the negation of the statement: "All data engineers use SQL" ($\forall x, P(x)$)?**
* **Answer:** There exists at least one data engineer who does not use SQL. 
* **Logic:** The negation of a Universal ("All") is an Existential ("At least one"), combined with the negation of the action.

**Q5. If $P$ is True and $Q$ is False, what is the truth value of $(\neg P \vee Q)$?**
* **Answer:** False 
* **Logic:** $\neg(\text{True}) \vee \text{False}$ becomes $\text{False} \vee \text{False}$, which evaluates to False.

**Q6. Is $\neg(P \wedge Q)$ equivalent to $(\neg P \vee \neg Q)$?**
* **Answer:** Yes 
* **Logic:** This is the exact definition of De Morgan's Law.

**Q7. Is $(P \wedge \neg P)$ a Tautology or a Contradiction?**
* **Answer:** Contradiction 
* **Logic:** A statement cannot be simultaneously True AND False. The result is always False.

**Q8. Negate the statement: $\forall x (P(x) \rightarrow Q(x))$.**
* **Answer:** $\exists x (P(x) \wedge \neg Q(x))$
* **Logic:** Flip the $\forall$ to an $\exists$. Then convert $P \rightarrow Q$ to $\neg P \vee Q$. Negating that using De Morgan's Law results in $P \wedge \neg Q$.
