## Sets and Power Sets

* **Union ($A \cup B$):** Combining all items. 
* **Intersection ($A \cap B$):** Only overlapping items.
* **Power Set $P(A)$:** The set of *all possible subsets*. Formula: $2^n$ elements. (To find *proper* subsets, subtract 1).

## Relations
* **Reflexive:** Every element relates to *itself* ($a, a$).
* **Symmetric:** If ($a, b$) exists, ($b, a$) *MUST* exist. (Mutual relationship).
* **Transitive:** If ($a, b$) and ($b, c$) exist, ($a, c$) *MUST* exist. (Inherited relationship).
* **Partial Order:** A relation that is Reflexive, *Antisymmetric* (if $A \subseteq B$ and $B \subseteq A$, then $A=B$), and Transitive.

## Functions
* **Injective (One-to-One):** Every $X$ maps to a *completely unique* $Y$. (No sharing seats).
* **Surjective (Onto):** Every $Y$ is mapped to by *at least one* $X$. (No empty targets).
* **Bijective:** Perfect *1:1 pairing*. Must be both Injective and Surjective. Requires sets of the exact same size.

## Algebraic Structures

**1. Groups (The 4 Core Properties)**
For a mathematical set and an operation (like addition or multiplication) to officially be called a "Group," it must pass four strict rules. If it fails even one, it is not a Group. 

* **Property 1: Closure.** If you take two items in the set and perform the operation, the answer must also be in the set.
  * *Example:* The set of All Integers under Addition (+). $5 + (-2) = 3$. Because $3$ is also an integer, it is Closed. (Odd integers fail this because $3 + 5 = 8$, and $8$ is not odd).
* **Property 2: Associativity.** The grouping of the operation does not matter.
  * *Rule:* $(a + b) + c = a + (b + c)$.
* **Property 3: Identity Element.** There must be a "do nothing" element in the set that leaves the original number unchanged. 
  * *Example:* For addition, the Identity is $0$ (because $5 + 0 = 5$). For multiplication, the Identity is $1$ (because $5 \times 1 = 5$).
* **Property 4: Inverse Element.** Every single item in the set must have a "partner" that cancels it out and returns it to the Identity state.
  * *Example:* In addition, the inverse of $5$ is $-5$ (because $5 + (-5) = 0$). 
* *Bonus Exam Term (Abelian Group):* If the group also passes the Commutative rule ($a + b = b + a$), it is called an **Abelian Group**.

**2. Lattices**
A Lattice is a specific type of hierarchy (a Partially Ordered Set) where every pair of elements has a unique upper bound and lower bound. 

* **The Join (Least Upper Bound):** The lowest possible point in the hierarchy that sits above both items.
* **The Meet (Greatest Lower Bound):** The highest possible point in the hierarchy that sits below both items.
* **The RBAC Analogy (Role-Based Access Control):** Imagine a cloud data platform's security hierarchy. 
  * You have two basic roles: **Data Reader** (can only view tables) and **Pipeline Runner** (can only trigger jobs). 
  * The **Join (Upper Bound)** is the **Data Engineer** role. The Data Engineer sits directly above them both because it inherits *both* reading and running permissions. 
  * The **Meet (Lower Bound)** is the **Guest** role. The Guest sits below them both, possessing only the base login permissions they all share.
  * The **Greatest Element (Top):** The **Super Admin** who oversees the entire architecture.
  * The **Least Element (Bottom):** A blocked or unauthenticated user with absolutely zero access.


## Practice Questions (Solved)
1. If Set $A$ has $5$ elements, how many elements are in its Power Set $P(A)$?
**Answer:** $32$. *($2^5 = 32$)*.

2. The number of proper subsets of a set with $3$ elements is:
**Answer:** $7$. *($2^3 = 8$. Subtract 1 for the set itself)*.

3. Let Set $A = \{1, 2, 3\}$. A relation $R$ on $A$ is defined as $R = \{(1, 1), (2, 2)\}$. Is this relation Reflexive?
**Answer:** No. *(It is missing the $(3, 3)$ mapping required for all elements)*.

4. The relation "is a subset of" ($\subseteq$) on the power set of any set is:
**Answer:** A Partial Order *(Reflexive, Antisymmetric, Transitive)*.

5. A function $f(x) = x^2$ mapping from Real Numbers to Real Numbers is:
**Answer:** Neither Injective nor Surjective. *(f(2) and f(-2) share an output; negative numbers are never hit)*.

6. If Set $A$ has $3$ elements and Set $B$ has $4$ elements, is it possible to create a Bijective function?
**Answer:** No. *(Sets must be the same size)*.

7. In a Boolean algebra, what is the dual of the expression: $x + (y \cdot z)$?
**Answer:** $x \cdot (y + z)$. *(Swap the AND/OR operators)*.

8. Which of the following sets forms a Group under standard addition?
**Answer:** The set of All Integers.
