## 1. Set Basics and Cardinality

A set is a well-defined collection of distinct objects.

* **Subset ($A \subseteq B$):** All elements of A are in B.
* **Proper Subset ($A \subset B$):** A is a subset of B, but $A \neq B$. 
    * Total subsets of a set with $n$ elements = $2^n$.
    * Total proper subsets = $2^n - 1$.
* **Power Set $P(A)$:** The set of all subsets of A.
    * Cardinality: $|P(A)| = 2^{|A|}$.
    * *Trap:* If $A = \{\emptyset\}$, $|A| = 1$. Therefore $|P(A)| = 2^1 = 2$.

## 2. Set Operations and Logic Equivalences

| Operation | Symbol | Definition | Logic Equivalent |
| :--- | :--- | :--- | :--- |
| **Union** | $A \cup B$ | Elements in A or B | OR ($\vee$) |
| **Intersection**| $A \cap B$ | Elements in both A and B| AND ($\wedge$) |
| **Difference** | $A - B$ | Elements in A but not B | $A \wedge \neg B$ |
| **Symmetric Diff**| $A \Delta B$ | Elements in A or B, but not both | XOR ($\oplus$) |

### Important Formulas
**1. Symmetric Difference:**

$$|A \Delta B| = |A| + |B| - 2|A \cap B|$$

**2. Inclusion-Exclusion (2 Sets):**

$$|A \cup B| = |A| + |B| - |A \cap B|$$

**3. Inclusion-Exclusion (3 Sets):**

$$|A \cup B \cup C| = |A| + |B| + |C| - (|A \cap B| + |B \cap C| + |A \cap C|) + |A \cap B \cap C|$$

## 3. Relations

A relation $R$ from set $A$ to set $B$ is a subset of the Cartesian Product $A \times B$. 

### Properties of a Relation on Set A
1.  **Reflexive:** $\forall a \in A$, $(a,a) \in R$. (Every element relates to itself).
2.  **Symmetric:** If $(a,b) \in R$, then $(b,a) \in R$. (Two-way streets only).
3.  **Transitive:** If $(a,b) \in R$ and $(b,c) \in R$, then $(a,c) \in R$. 
4.  **Anti-Symmetric:** If $(a,b) \in R$ and $(b,a) \in R$, then $a = b$. (No two-way streets between different elements).

*Note: An **Equivalence Relation** is Reflexive, Symmetric, and Transitive.*

### Relation Counting Formulas (For a set of size $n$)
* **Total Relations:** $2^{n^2}$
* **Reflexive Relations:** $2^{n^2 - n}$ (Derived via the Matrix Method)
* **Symmetric Relations:** $2^{\frac{n(n+1)}{2}}$

## 4. Functions (Mappings) & The Database Analogy

A function $f: A \to B$ maps every element in the domain $A$ to exactly one element in the co-domain $B$. 

> **Real-World Analogy:** Think of Set $A$ as a **Source Table** (e.g., `Employees`) and Set $B$ as a **Target Table** (e.g., `Departments`). A function is a complete mapping where every Employee is assigned to exactly one Department. 

* **Total Functions:** $|B|^{|A|}$ (For every employee in $A$, you have $|B|$ choices).
* **The Cross Join Trap:** Do not confuse functions with a Cartesian Product ($A \times B$). A cross join gives $|A| \times |B|$ records, but a function is the total number of unique organizational charts you can build ($|B|^{|A|}$).

### Types of Functions
1.  **One-to-One (Injective):** No two elements in $A$ map to the same element in $B$. (Unique outputs).
    * *Analogy:* Mapping `Employees` to `Assigned_Laptops`. If Emp 1 gets Laptop A, Emp 2 cannot get Laptop A.
    * *Condition:* $|A| \leq |B|$
2.  **Onto (Surjective):** Every element in $B$ is mapped to by at least one element in $A$. (No orphans in target).
    * *Analogy:* Mapping `Employees` to `Project_Teams`. A team is only valid if at least one employee is mapped to it. No team is left empty.
    * *Condition:* $|A| \geq |B|$
    * *Formula (if $|B| = 2$):* $2^{|A|} - 2$
3.  **Bijection:** Both One-to-One and Onto.
    * *Analogy:* A perfect Primary-Key to Primary-Key mapping.
    * *Condition:* $|A| = |B|$
    * *Note:* A function must be a bijection to have an **Inverse**.

## 5. Advanced Combinatorics Traps

**The Nested Subset Problem:**
How many pairs of subsets $(A, B)$ can be formed from a set $S$ (size $n$) such that $A \subseteq B \subseteq S$?
* **Formula:** $3^n$
* **The Layered Filtering Analogy:** Think of this as filtering data layers. Look at any single element $x$ in the master set $S$. For $A \subseteq B$ to be valid, $x$ can only exist in exactly 3 states:
    1. It is in **both** A and B.
    2. It is **only** in B.
    3. It is in **neither** A nor B.
    *(It can never be only in A, because A is restricted inside B).* Since every element has 3 independent paths, the combinations are $3^n$.

## 6. Deconstructed Problem Bank

### Category A: Sets, Subsets & Cardinality

**Problem 1: Power Sets**
* **Question:** If $S = \{1, 2, \{3, 4\}\}$, how many elements are in $P(S)$?
* **Solution:** The elements are $1, 2,$ and the set $\{3, 4\}$. Cardinality is 3. $|P(S)| = 2^3 = \mathbf{8}$.

**Problem 2: The Element Trap**
* **Question:** If $A=\{1, 2\}$, which is TRUE? (A) $\{1\} \in P(A)$ (B) $1 \in P(A)$
* **Solution:** $P(A)$ contains sets, not raw numbers. Since $\{1\} \subseteq A$, it follows that $\{1\} \in P(A)$. **(A) is true**.

**Problem 3: Nested Power Sets**
* **Question:** If $A = \{\emptyset\}$, what is $|P(P(A))|$?
* **Solution:** $|A| = 1$. $|P(A)| = 2^1 = 2$. $|P(P(A))| = 2^2 = \mathbf{4}$.

**Problem 4: Advanced Nested Power Sets**
* **Question:** Let set $A = \{x, \{y, z\}\}$. What is the total number of elements in $P(P(A))$?
* **Solution:** $A$ contains two distinct elements: $x$ and the subset $\{y,z\}$. $|A|=2$. $|P(A)| = 2^2 = 4$. Finally, $|P(P(A))| = 2^4 = \mathbf{16}$.

### Category B: Set Operations & Inclusion-Exclusion

**Problem 5: De Morgan's for Sets**
* **Question:** Simplify $A - (B \cup C)$.
* **Solution:** $A \cap \neg(B \vee C) \equiv A \cap (\neg B \wedge \neg C) \equiv (A - B) \cap (A - C)$.

**Problem 6: Set Logic Simplification**
* **Question:** Simplify $(A - B) \cup (A \cap B)$.
* **Solution:** "Elements in A but not B, combined with elements in both A and B." This reunites the split pieces of A. The answer is simply **$A$**.

**Problem 7: 3-Set Inclusion-Exclusion**
* **Question:** Survey of 60 people: 25 read H, 26 read T, 26 read I. 9 read H&I, 11 read H&T, 8 read T&I. 3 read all. How many read at least one?
* **Solution:** Apply the formula: $(25 + 26 + 26) - (9 + 11 + 8) + 3 = 77 - 28 + 3 = \mathbf{52}$.

**Problem 8: Symmetric Difference Cardinality**
* **Question:** $|A| = 10, |B| = 20, |A \cap B| = 5$. Find $|A \Delta B|$.
* **Solution:** Formula: $|A| + |B| - 2|A \cap B|$. $10 + 20 - 2(5) = \mathbf{20}$.

### Category C: Relations & Properties

**Problem 9: Identifying Equivalence**
* **Question:** $R = \{(1, 1), (2, 2), (3, 3), (1, 2), (2, 1)\}$ on $A=\{1, 2, 3\}$. Type of relation?
* **Solution:** Reflexive (all diagonals present), Symmetric (1,2 and 2,1 are mirrors), Transitive (1,2 and 2,1 yields 1,1). It is an **Equivalence Relation**.

**Problem 10: The Transitive Trap**
* **Question:** $R = \{(1,1), (2,2), (3,3), (4,4), (1,2), (2,1), (2,3), (3,2)\}$. Why is it not an Equivalence Relation?
* **Solution:** It fails Transitivity. We have $(1,2)$ and $(2,3)$, but the pair **$(1,3)$ is missing**.

**Problem 11: Anti-Symmetry**
* **Question:** Which is Anti-symmetric? $R = \{(1,2), (1,1)\}$ or $R = \{(1,2), (2,3), (1,3)\}$?
* **Solution:** Anti-symmetry forbids two-way pairs (unless $a=b$). Neither set has a two-way pair (like 1,2 and 2,1). Therefore, **Both** are anti-symmetric.

**Problem 12: Cartesian Power Trap**
* **Question:** Relation $R = A \times A$. Properties?
* **Solution:** This is the Universal Relation containing every possible pair. It cannot fail any rule. It is **Reflexive, Symmetric, and Transitive**.

**Problem 13: Counting Reflexive Relations**
* **Question:** Reflexive relations on a set of $n$ elements?
* **Solution:** Using the matrix method, diagonals are fixed (1 choice). Remaining cells are $n^2 - n$, each with 2 choices. Total = **$2^{n^2 - n}$**.

### Category D: Functions & Mappings

**Problem 14: Function Counting**
* **Question:** Set X has 5 elements, Set Y has 3 elements. Total distinct functions?
* **Solution:** Target^(Source) = $|Y|^{|X|} = 3^5 = \mathbf{243}$.

**Problem 15: Counting Onto Functions**
* **Question:** $|A|=4, |B|=2$. How many Onto functions?
* **Solution:** Total functions = $2^4 = 16$. Failures (where everything maps to just 1 element) = 2. Onto = $16 - 2 = \mathbf{14}$.

**Problem 16: One-to-One and Domain Trap**
* **Question:** $f(x) = x^2$ for $f: \mathbb{R} \to \mathbb{R}$. Is it One-to-One?
* **Solution:** No. Both $f(2)$ and $f(-2)$ map to $4$. Two sources map to the same target.

**Problem 17: Range & Co-domain**
* **Question:** $f: \mathbb{Z} \to \mathbb{Z}$ defined by $f(n) = 2n$. What is the Range?
* **Solution:** The mapping multiplies integers by 2. The range is **All Even Integers** (not all positive, because domain includes negative integers).

**Problem 18: Inverses and Bijections**
* **Question:** Does $f(x) = 2x + 3$ have an inverse?
* **Solution:** Yes. It is a straight line, representing a perfect Bijection. Every unique Y can be traced back to a unique X via $x = (y - 3) / 2$.

**Problem 19: Function Composition**
* **Question:** $f(x) = x + 2$, $g(x) = x^2$. Find $g(f(x))$.
* **Solution:** Pass the entire output of $f$ into $g$. $g(x+2) = \mathbf{(x+2)^2}$.

**Problem 20: Equivalence Classes**
* **Question:** $a \equiv b \pmod 3$. How many distinct classes?
* **Solution:** Think of these as "Logical Buckets". Dividing any integer by 3 leaves remainders of 0, 1, or 2. Therefore, there are exactly **3** distinct equivalence classes.
