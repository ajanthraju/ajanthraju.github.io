# 50-Day Computer Science & Information Technology Study Planner

### **Week 1: Engineering Mathematics (Days 1–7)**
* **Day 1: Logic & Sets**
    * Mathematical Logic: Propositional and first-order logic.
    * Set Theory: Sets, relations, functions.
* **Day 2: Abstract Algebra & Basic Combinatorics**
    * Algebra: Partial orders and lattices; Monoids, Groups; Boolean Algebra.
    * Combinatorics Part 1: Counting, permutations, combinations.
* **Day 3: Advanced Combinatorics**
    * Combinatorics Part 2: Summation, recurrence relations, generating functions, asymptotics.
* **Day 4: Graph Theory**
    * Connectivity, matching, coloring, topological sorting.
    * Dijkstra’s Algorithm, Minimal Spanning Tree (MST), DFS.
    * Biconnectivity, Euler Circuits, and applications.
* **Day 5: Linear Algebra**
    * Matrices, determinants, system of linear equations.
    * Eigenvalues and eigenvectors, LU decomposition.
* **Day 6: Calculus**
    * Limits, continuity, and differentiability; Mean value theorems.
    * Theorems of integral calculus, evaluation of definite and improper integrals.
    * Maxima and minima; Partial and total derivatives.
* **Day 7: Probability, Statistics & Numerical Methods**
    * Random variables; Uniform, normal, exponential, Poisson, and binomial distributions.
    * Mean, median, mode, and standard deviation; Conditional probability and Bayes theorem.
    * Numerical Methods: Solutions of non-linear algebraic equations (Secant, Bisection, and Newton-Raphson methods); Numerical integration (Trapezoidal and Simpson’s rules).

---

### **Week 2: Digital Logic & Computer Architecture (Days 8–12)**
* **Day 8: Digital Logic Basics**
    * Logic functions, Boolean algebra, minimization.
    * Combinational and sequential circuits (design and synthesis).
* **Day 9: Data Representation & HDL**
    * Number representations and computer arithmetic (fixed and floating point).
    * Hardware Description Language (HDL) for circuits.
* **Day 10: Computer Organization**
    * Machine instructions and addressing modes.
    * ALU, data-path, and control unit (Single-cycle, Multi-cycle, Micro-programming, and Hard-wired).
    * Behavioral HDL description of systems; Exception handling.
* **Day 11: Pipelining**
    * Pipelined MIPS data path.
    * Pipeline hazards (Structural, Control, Data); Hazard detection and resolution; Pipelining control.
* **Day 12: Memory & I/O Systems**
    * Memory hierarchy (Cache, Main, and Secondary storage); SRAM and DRAM design; Virtual memory.
    * I/O interface (Interrupt and DMA mode); I/O device characteristics, buses, and bus arbitration.
    * Processor/OS interface.

---

### **Week 3: Programming, Data Structures & Algorithms (Days 13–19)**
* **Day 13: Programming Fundamentals**
    * Programming in C; Recursion; Abstract data types.
* **Day 14: Linear Data Structures**
    * Arrays, stacks, queues, linked lists.
* **Day 15: Non-Linear Data Structures**
    * Trees: Binary Search Trees, AVL Trees, Red-Black Trees, Splay Trees.
    * Heaps: Binary heaps, Leftist heaps.
* **Day 16: Basic Algorithms**
    * Searching.
    * Sorting Part 1: Insertion, Merge, Quick, Heap.
* **Day 17: Advanced Sorting & Hashing**
    * Sorting Part 2: Disk-based, k-way merging.
    * Hashing techniques.
* **Day 18: Complexity & Analysis**
    * Asymptotic analysis (best, worst, average cases) of time and space complexity; Upper and lower bounds.
    * Complexity Theory: Concepts of complexity classes (P, NP, NP-hard, NP-complete).
* **Day 19: Algorithm Design Techniques**
    * Greedy, dynamic programming, divide-and-conquer, backtracking, branch and bound.

---

### **Week 4: Theory of Computation & Compiler Design (Days 20–24)**
* **Day 20: Finite Automata**
    * Regular expressions and finite automata (NFA, DFA, Minimal FA); Kleene’s Theorem.
* **Day 21: Grammars & Languages**
    * Context-free grammars and push-down automata.
    * Regular and context-free languages, pumping lemma.
* **Day 22: Turing Machines & Computability**
    * Turing machines, recursively enumerable languages, undecidability, and unsolvable problems.
* **Day 23: Compiler Front-End**
    * Lexical analysis, parsing, syntax-directed translation; Runtime environments.
* **Day 24: Compiler Back-End & Optimization**
    * Intermediate and target code generation.
    * Optimization: Local optimization, data flow analyses (constant propagation, liveness analysis, common sub-expression elimination), basics of code optimization.

---

### **Week 5: Operating Systems & System Programming (Days 25–29)**
* **Day 25: OS Fundamentals & Concurrency**
    * System calls, processes, threads.
    * Inter-process communication, concurrency, and synchronization.
* **Day 26: Deadlocks & Scheduling**
    * Deadlock (detection and prevention).
    * CPU and I/O scheduling; Disk scheduling and management.
* **Day 27: Memory & Storage Management**
    * Memory management and virtual memory.
    * File systems, free-space management, swap-space management; Protection and security.
* **Day 28: OS Case Studies & Assembly**
    * Case Studies: Design principles of Linux and Windows 7.
    * System Programming: Elements of Assembly Language; Design of single and two-pass assemblers.
* **Day 29: System Software Tools**
    * Macros and macro processors; Linkers and Loaders.
    * Software Tools: Editors, debug monitors, programming environments.

---

### **Week 6: Database Systems (Days 30–34)**
* **Day 30: Data Modeling & Algebra**
    * ER-model, Relational model.
    * Relational algebra, tuple calculus.
* **Day 31: SQL & DB Design**
    * SQL (Data definition, Queries, Updates, Views).
    * Integrity constraints, functional dependencies, and normalization.
* **Day 32: Advanced Database Design**
    * Relational database design.
    * Design of Object-oriented Databases.
* **Day 33: Storage, Indexing & Querying**
    * Record storage, file organization (Heap, Sorted, Hashing).
    * Indexing (B and B+ trees); Query processing.
* **Day 34: Transaction Management**
    * Concurrency control (Two-phase locking), serializability, and schedules; Deadlock.
    * Recovery techniques (Immediate/Deferred update, Shadow paging).

---

### **Week 7: Computer Networks (Days 35–40)**
* **Day 35: Network Fundamentals**
    * Concept of layering (OSI and TCP/IP stacks).
    * Basics of packet, circuit, and virtual circuit-switching; Hubs, switches, gateways, and routers; LAN technologies (Ethernet, Token ring).
* **Day 36: Data Link Layer**
    * Framing, error detection/correction.
    * Medium Access Control, Ethernet bridging.
* **Day 37: Network Layer & Routing**
    * Routing protocols (Shortest path, flooding, distance vector, link state).
    * Fragmentation, IP addressing (IPv4, CIDR, NAT); IP support protocols (ARP, DHCP, ICMP).
* **Day 38: Transport & Application Layers**
    * Flow and congestion control; UDP, TCP, sockets.
    * Protocols: DNS, SMTP, POP, HTTP, FTP, Email.
* **Day 39: High Performance Networks & Security**
    * ISDN, BISDN, ATM, Frame relay, MPLS, Optical Networks.
    * Network Security: Public/Private key cryptography, digital signatures, firewalls.
* **Day 40: Wireless Adhoc Networks**
    * Routing (AODV, DSR, DSDV, ODMRP), Multi-clustering, and Power issues.

---

### **Week 8: Software Engineering (Days 41–44)**
* **Day 41: Software Paradigms**
    * Life cycle models (Waterfall, Incremental, Spiral, WINWIN Spiral, Evolutionary, Prototyping, Object-oriented).
* **Day 42: Project Management**
    * Project planning, risk analysis, scheduling, tracking.
    * Software quality assurance (SQA), configuration management.
* **Day 43: Requirements & Design**
    * Rapid prototyping techniques.
    * Real-time systems and software design.
* **Day 44: Testing & Metrics**
    * Strategic approach to testing, types of software testing.
    * Software metrics.

---

### **Week 9: Computer Graphics & Multimedia (Days 45–47)**
* **Day 45: 2D/3D Graphics & Transformations**
    * Line, Curve, and Ellipse drawing algorithms.
    * 2D and 3D Geometric transformations, clipping, viewing, and modeling.
* **Day 46: Advanced Graphics & Multimedia Basics**
    * Color models; Animation.
    * Multimedia: Elements, architecture, and evolving technologies.
* **Day 47: Multimedia Standards & Compression**
    * Multimedia Data Interface Standards; Multimedia Databases.
    * Compression: Binary image, Color/Gray scale, Video, Audio, and Fractal compression; Virtual Reality design.

---

### **Week 10: Web Technologies & E-Commerce (Days 48–50)**
* **Day 48: Web Concepts & Basic Client-Side**
    * World Wide Web, Web servers/browsers, URL, MIME, HTTP, SGML; Internet protocols and standards.
    * Client-Side Part 1: HTML forms, CGI concepts.
* **Day 49: Advanced Client-Side & Basic Server-Side**
    * Client-Side Part 2: Applets, JavaScript, Dynamic HTML, ActiveX controls, Client-side scripting.
    * Server-Side Part 1: Servlets, JSP.
* **Day 50: Advanced Server-Side & E-Commerce**
    * Server-Side Part 2: Session management, Cookies, Database access (SQL) through web architecture.
    * E-Commerce: Business models, E-marketing, E-security, E-payment systems, E-CRM.
