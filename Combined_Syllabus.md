# Combined Computer Science & Information Technology Syllabus

## 1. Engineering Mathematics
* **Mathematical Logic:** Propositional and first-order logic.
* **Set Theory & Algebra:** Sets, relations, functions; Partial orders and lattices; Monoids, Groups; Boolean Algebra.
* **Combinatorics:** Counting, permutations, combinations, summation, recurrence relations, generating functions, asymptotics.
* **Graph Theory:** Connectivity, matching, coloring, topological sorting, Dijkstra’s Algorithm, Minimal Spanning Tree (MST), DFS, Biconnectivity, Euler Circuits, applications.
* **Linear Algebra:** Matrices, determinants, system of linear equations, eigenvalues and eigenvectors, LU decomposition.
* **Calculus:** Limits, continuity, and differentiability; Mean value theorems; Theorems of integral calculus, evaluation of definite and improper integrals; Maxima and minima; Partial and total derivatives.
* **Probability and Statistics:** Random variables; Uniform, normal, exponential, Poisson, and binomial distributions; Mean, median, mode, and standard deviation; Conditional probability and Bayes theorem.
* **Numerical Methods:** Numerical solutions of non-linear algebraic equations (Secant, Bisection, and Newton-Raphson methods); Numerical integration (Trapezoidal and Simpson’s rules).

---

## 2. Digital Logic & Computer Architecture
* **Digital Logic:** Logic functions, Boolean algebra, minimization; Combinational and sequential circuits (design and synthesis); Number representations and computer arithmetic (fixed and floating point); Hardware Description Language (HDL) for circuits.
* **Computer Organization:** Machine instructions and addressing modes; ALU, data-path, and control unit (Single-cycle, Multi-cycle, Micro-programming, and Hard-wired); Behavioral HDL description of systems; Exception handling.
* **Pipelining:** Pipelined MIPS data path; Pipeline hazards (Structural, Control, Data); Hazard detection and resolution; Pipelining control.
* **Memory & I/O:** Memory hierarchy (Cache, Main, and Secondary storage); SRAM and DRAM design; Virtual memory; I/O interface (Interrupt and DMA mode); I/O device characteristics, buses, and bus arbitration; Processor/OS interface.

---

## 3. Programming, Data Structures & Algorithms
* **Programming & Data Structures:** Programming in C; Recursion; Abstract data types; Arrays, stacks, queues, linked lists; Trees (Binary Search Trees, AVL Trees, Red-Black Trees, Splay Trees); Binary heaps, Leftist heaps.
* **Algorithms:** Searching, sorting (Insertion, Merge, Quick, Heap, Disk-based, k-way merging), hashing.
* **Design & Analysis:** Asymptotic analysis (best, worst, average cases) of time and space complexity; Upper and lower bounds.
* **Techniques:** Greedy, dynamic programming, divide-and-conquer, backtracking, branch and bound.
* **Complexity Theory:** Concepts of complexity classes (P, NP, NP-hard, NP-complete).

---

## 4. Theory of Computation & Compiler Design
* **Theory of Computation:** Regular expressions and finite automata (NFA, DFA, Minimal FA); Kleene’s Theorem; Context-free grammars and push-down automata; Regular and context-free languages, pumping lemma; Turing machines, recursively enumerable languages, undecidability, and unsolvable problems.
* **Compiler Design:** Lexical analysis, parsing, syntax-directed translation; Runtime environments; Intermediate and target code generation.
* **Optimization:** Local optimization, data flow analyses (constant propagation, liveness analysis, common sub-expression elimination), basics of code optimization.

---

## 5. Operating Systems & System Programming
* **Operating Systems:** System calls, processes, threads, inter-process communication, concurrency, and synchronization; Deadlock (detection and prevention).
* **Management:** CPU and I/O scheduling; Disk scheduling and management; Memory management and virtual memory; File systems, free-space management, swap-space management; Protection and security.
* **Case Studies:** Design principles of Linux and Windows 7.
* **System Programming:** Elements of Assembly Language; Design of single and two-pass assemblers; Macros and macro processors; Linkers and Loaders.
* **Software Tools:** Editors, debug monitors, programming environments.

---

## 6. Database Systems
* **Modeling:** ER-model, Relational model; Relational algebra, tuple calculus, SQL (Data definition, Queries, Updates, Views).
* **Design:** Integrity constraints, functional dependencies, and normalization; Relational database design; Design of Object-oriented Databases.
* **Storage & Querying:** Record storage, file organization (Heap, Sorted, Hashing); Indexing (B and B+ trees); Query processing.
* **Transactions:** Concurrency control (Two-phase locking), serializability, and schedules; Deadlock; Recovery techniques (Immediate/Deferred update, Shadow paging).

---

## 7. Computer Networks
* **Fundamentals:** Concept of layering (OSI and TCP/IP stacks); Basics of packet, circuit, and virtual circuit-switching; Hubs, switches, gateways, and routers; LAN technologies (Ethernet, Token ring).
* **Data Link & Network Layers:** Framing, error detection/correction; Medium Access Control, Ethernet bridging; Routing protocols (Shortest path, flooding, distance vector, link state); Fragmentation, IP addressing (IPv4, CIDR, NAT); IP support protocols (ARP, DHCP, ICMP).
* **Transport & Application Layers:** Flow and congestion control; UDP, TCP, sockets; Protocols: DNS, SMTP, POP, HTTP, FTP, Email.
* **High Performance & Wireless:** ISDN, BISDN, ATM, Frame relay, MPLS, Optical Networks. Wireless Adhoc Networks: Routing (AODV, DSR, DSDV, ODMRP), Multi-clustering, and Power issues.
* **Network Security:** Public/Private key cryptography, digital signatures, firewalls.

---

## 8. Software Engineering
* **Paradigms:** Life cycle models (Waterfall, Incremental, Spiral, WINWIN Spiral, Evolutionary, Prototyping, Object-oriented).
* **Management:** Project planning, risk analysis, scheduling, tracking, software quality assurance (SQA), configuration management.
* **Requirements & Design:** Rapid prototyping techniques; Real-time systems and software design.
* **Testing & Metrics:** Strategic approach to testing, types of software testing, software metrics.

---

## 9. Computer Graphics & Multimedia
* **Graphics:** Line, Curve, and Ellipse drawing algorithms; 2D and 3D Geometric transformations, clipping, viewing, and modeling; Color models; Animation.
* **Multimedia:** Elements, architecture, and evolving technologies; Multimedia Data Interface Standards; Multimedia Databases.
* **Compression:** Binary image, Color/Gray scale, Video, Audio, and Fractal compression; Virtual Reality design.

---

## 10. Web Technologies & E-Commerce
* **Web Concepts:** World Wide Web, Web servers/browsers, URL, MIME, HTTP, SGML; Internet protocols and standards.
* **Client-Side:** HTML forms, CGI concepts, Applets, JavaScript, Dynamic HTML, ActiveX controls, Client-side scripting.
* **Server-Side:** Servlets, JSP, Session management, Cookies, Database access (SQL) through web architecture.
* **E-Commerce:** Business models, E-marketing, E-security, E-payment systems, E-CRM.
